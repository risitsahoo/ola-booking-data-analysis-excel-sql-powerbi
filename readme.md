## Project Overview

The **OLA Booking Data Analysis Project** is an end-to-end **Business Intelligence and Data Analytics solution** developed to analyze ride-hailing operations and transform raw booking data into actionable business insights.

The project integrates **SQL-based analysis**, **Power BI dashboards**, and **DAX measures** to deliver real-time decision-support analytics for stakeholders.

---

## Business Problem Statement

Ride-hailing platforms face challenges such as high cancellation rates, driver availability issues, uneven vehicle utilization, and fluctuating revenue.

Without centralized analytics, identifying root causes becomes difficult.  
This project solves that gap using structured data analysis and interactive dashboards.

---

## Business Objectives

- Analyze overall booking performance  
- Identify success and cancellation trends  
- Understand cancellation reasons  
- Evaluate vehicle demand and utilization  
- Analyze customer and driver ratings  
- Track revenue and payment behavior  
- Identify high-value customers  
- Build executive-level BI dashboards  

---

## Dataset Overview

- Source: Excel (.xlsx)  
- Period: July 2024  
- Each record represents a single booking transaction

### Key Fields

- Booking ID  
- Booking Status  
- Vehicle Type  
- Customer & Driver Ratings  
- Ride Distance  
- Booking Value  
- Payment Method  
- Cancellation & Incomplete Ride Reasons

## 15. Key Insights & Findings

- 100K+ bookings analyzed  
- Cancellation rate approx. 28%  
- UPI is the most preferred payment method  
- Premium vehicles generate higher revenue  
- Customer cancellations exceed driver cancellations  
- Ratings remain stable around 4.0
